# Endpoint Backlog

Track datasets we plan to expose as new API routes. Each item lists the intended path plus a quick note on the data source or purpose.

- [ ] `/api/holidays` — Official Phil. holidays with working-day flags.
- [ ] `/api/government-offices` — Agency directory with contact info and codes.
- [ ] `/api/public-transit` — MRT/LRT/PNR lines, stations, and service hours.
- [ ] `/api/disaster-alerts` — PAGASA/PHIVOLCS alert levels for automation.
- [ ] `/api/business-permits` — Common LGU permit requirements and fees.
- [ ] `/api/zipcodes` — Zip/postal codes mapped to cities and barangays.
- [ ] `/api/benefits` — Gov assistance programs with eligibility references.
- [ ] `/api/language-codes` — Major PH languages with ISO codes and regions.
- [ ] `/api/birth-certificate` — PSA document checklist and processing timers.
- [ ] `/api/tax-tables` — BIR tax brackets and withholding tables per year.
- [ ] `/api/housing-projects` — DHSUD/SHFC socialized housing inventory and status.
- [ ] `/api/philhealth-premiums` — Contribution tables by salary grade and membership.
- [ ] `/api/road-closures` — MMDA/DPWH advisories for traffic rerouting.
- [ ] `/api/school-calendar` — DepEd/CHED academic calendars and key deadlines.
- [ ] `/api/weather-stations` — Metadata for DOST weather stations with coordinates.
- [ ] `/api/visa-requirements` — DFA visa categories and documentary requirements.
- [ ] `/api/sme-support` — DTI/DTI Negosyo Center programs with contact points.
- [ ] `/api/agri-prices` — DA price watch for staple crops and livestock.
- [ ] `/api/port-schedules` — Major seaport routes, operators, and sailing times.
- [ ] `/api/election-data` — COMELEC precinct data and candidate directories.
